claude-local-starter

Portable Claude Code setup. Clone and run once — installs skills, plugins, MCP servers, shell functions, and CLAUDE.md on any machine.

Quick start

git clone https://github.com/indranildchandra/claude-local-starter
bash claude-local-starter/install.sh

Installer modes

bash install.sh                  # safe update (default)
bash install.sh --clean-install  # full overwrite (backs up first)
bash install.sh --dry-run        # preview, no changes written

Measure token costs

Use token-audit.sh to measure the exact token footprint of any skill or plugin before committing to it.

1. Save state & enable everything

bash token-audit.sh --save

2. Reload inside Claude Code

/reload-plugins

3. Check token footprint

/context

Each loaded skill and plugin appears with its token count. Note the number for any item you are evaluating.

4. Restore your original state

bash token-audit.sh --restore

5. Apply restore in Claude Code

/reload-plugins

Everything returns to your pre-audit state. Only keep a skill or plugin enabled if the token cost is worth the benefit.

Extending

  • Add skills → skills/your-skill/SKILL.md in repo, run bash install.sh
  • Add commands → commands/your-cmd.md in repo, run bash install.sh
  • Edit global config → claude-md-master/CLAUDE.md, run bash install.sh
